# 🌐 Responsive-Landing-Page

A clean and responsive static website built using **HTML5** and **CSS3**. This project focuses on creating a modern, visually appealing user interface while practicing the fundamentals of frontend web development.

## 🚀 Live Demo

👉 https://samyy-110.github.io/Simple-Website/


---

## ✨ Features

* 🎨 Modern and clean UI
* 📱 Responsive layout
* 🖱️ Smooth hover effects
* 🧭 Simple and intuitive navigation
* ⚡ Fast loading static website
* 💻 Built entirely with HTML and CSS

---

## 🛠️ Tech Stack

* HTML5
* CSS3

## 💻 Getting Started

Clone the repository:

```bash
git clone https://github.com/samyy-110/Simple-Website.git
```

Open the project folder:

```bash
cd Simple-Website
```

Run the project by opening `index.html` in your preferred browser.
